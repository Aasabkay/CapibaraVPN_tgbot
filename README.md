# 🦦 CapibaraVPN Bot

Асинхронный Telegram-бот на базе **Aiogram 3** для управления приватным VPN-сервисом. Бот поддерживает систему рассылок, логирование жалоб пользователей и использует **Redis** в качестве отказоустойчивого хранилища состояний (FSM).

---

## 🚀 Основной функционал

* **Управление подписками:** Выдача и обновление конфигураций VPN (VLESS/Reality).
* **Админ-панель:** Встроенная мультиконтейнерная рассылка сообщений пользователям с защитой от флуда (`Rate Limits`).
* **Обратная связь:** Система тикетов/жалоб для пользователей с автоматической пересылкой медиафайлов и логов админу.
* **Стабильность:** Изолированная архитектура на базе Docker-контейнеров.

---

## 🛠 Технологический стек

* **Язык:** Python 3.11+
* **Фреймворк:** Aiogram 3.x (Asyncio)
* **База данных / Хранилище:** Redis 7 (State Management & Storage)
* **Оркестрация:** Docker, Docker Compose

---
