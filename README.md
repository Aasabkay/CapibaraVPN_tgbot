# 🦦 CapibaraVPN Bot

Асинхронный Telegram-бот на базе **Aiogram 3** для управления приватным VPN-сервисом с установленной 3X-UI панелью. Бот поддерживает систему рассылок, логирование жалоб пользователей и использует **Redis** в качестве отказоустойчивого хранилища состояний (FSM).

---

## 🚀 Основной функционал

* **Управление ключами:** Обновление конфигураций VPN (VLESS/Reality).
* **Админ-панель:** Встроенная мультиконтейнерная рассылка сообщений пользователям с защитой от флуда (`Rate Limits`).
* **Обратная связь:** Система тикетов/жалоб для пользователей с автоматической пересылкой медиафайлов и логов админу.
* **Стабильность:** Изолированная архитектура на базе Docker-контейнеров.

---

## 🛠 Технологический стек

* **Язык:** Python 3.11+
* **Фреймворки:** Aiogram 3.x (Asyncio), httpx
* **База данных / Хранилище:** Redis 7 (State Management & Storage)
* **Оркестрация:** Docker, Docker Compose

---
